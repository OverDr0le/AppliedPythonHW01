Репозиторий состоит из 2з частей:

2. [Streamlit веб-приложение](app/app.py) 

# Как запустить
Сервис доступен по [ссылке](https://openweatherhw.streamlit.app/)
## Если хочется локально
1. Создаём виртуальное окружение python3.12 и скачиваем нужные библиотеки `poetry install`
2. Запускаем в директории /app `streamlit run app.py`

__Note__ CSS для оформления дизайна веб-приложения был взят у Gemini 3.
